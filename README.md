## keybindings

`SUPER + Q` - terminal

`SUPER + R` - launcher

`SUPER + W` - browser

`SUPER + C` - close window

`SUPER + E` - file-manager

`SUPER + N` - activate nightshift

`SUPER + L` - lock system

`SUPER + SHIFT + R` - reload waybar

`SUPER + SHIFT + G` - copy/paste dotfiles changes into local repo

`SUPER + SHIFT + W` - waybar on/off
